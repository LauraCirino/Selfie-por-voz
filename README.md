E WebApp que tira fotos por comando de voz, basta dizer a palavra "selfie" e aguardar 5 segundo que a foto ira para a pasta de Downloads

---
## 📚 Aprendizados
- Machine Learning
- Processamento de linguagem natural
- API de voz
- bootstrap
- webcam.js

## ⚙️ Funcionalidades
- clique no botão iniciar e diga "selfie", aguarde 5 segundos para selfie ser tirada e feito o download

## 🛠️ Stack
- Bootstrap
- Webcam.js
- p5.js
- ml5.js
- JavaScript
- HTML5
- CSS3

## 🚀 Deploy
1. Clone ou baixe este repositório.

```bash
git clone https://github.com/cigarraa/Selfie-por-voz
```

2. Abra o arquivo `index.html` em seu navegador.
